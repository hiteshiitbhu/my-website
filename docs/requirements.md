# Requirements

## Technical stack
- Astro v6.x (latest)
- Node.js v22+
- Markdown/mdx for content/blog
- PostCSS, CSS modules or TailwindCSS for styles (TBD during initial design)

## Must-Haves
- Bold/dynamic visual design, matching client spec
- Responsive, accessible, SEO optimized
- Contact section with direct reach options (email + social links)
- Project showcase with interactive/animated UI
- Dark/light mode toggle
- Documentation and dev notes in `/docs`

## Nice-to-Haves
- Blog (markdown, commenting, social preview)
- Animations/motion (framer-motion or GSAP if React/JS interop needed)
- Analytics/telemetry, privacy-friendly
