# Development Plan

This plan tracks each stage of developing the Astro-based bold personal portfolio.

## Milestones
1. **Setup & Initialization**  ✅
   - Project structure, docs created, Astro/scaffold done, Git initialized
2. **Design System & Theme**
   - Implement bold/dynamic color palette & typography
   - Set up global styles and fonts
3. **Navigation, Layout & Routing**
   - Header, sticky navbar, accessible links
   - Scroll-into-view & transitions
4. **Page Section Implementation**
   - Home (hero, tagline, links)
   - About (bio, skills, timeline)
   - Projects (cards/gallery, modal/details)
   - Contact (form with email)
5. **Advanced Features**
   - Animations, motion, reveal-on-scroll
   - Dark/light mode toggle
   - Blog scaffolding (later phase; MD/MDX)
   - SEO, meta tags, structured data
6. **Testing & QA**
   - Responsive, accessibility (a11y), cross-browser
7. **Launch & Docs Update**
