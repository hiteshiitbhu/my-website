# Project Structure

This document describes the layout and purpose of each directory and file in the Astro personal portfolio project.

## Root Directories

- `/src` — Source files for pages, layouts, components, and styles
- `/public` — Static assets (images, favicon, robots.txt, etc.)
- `/docs` — All documentation and planning for development
- `astro.config.mjs`, `tsconfig.json`, `package.json` — Configuration and project metadata
- `/README.md` — Main project overview

## Pages/Sections
- **Home:** Hero, tagline, availability, social/contact links
- **About:** Bio, professional summary, story
- **Projects:** Portfolio/gallery of work, details modal
- **Contact:** Email + social links, alternate ways to connect
- **Blog (Future optional):** Markdown/MDX-backed blogging structure

## Components
- Navbar
- Footer
- Animated heading/hero
- Project cards
- Contact form
- Contact links block
- Dark mode/theme toggle
