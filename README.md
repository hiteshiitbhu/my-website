# Hitesh Bansal Portfolio (Astro)

Single-page, high-animation personal portfolio built with Astro.

## Sections
- Home
- About
- Skills
- Projects
- Career Roadmap
- Contact (email + social links)

## Highlights
- Bold experimental design system with layered animated backgrounds
- Staggered reveal animations and interactive project cards
- Dark/light theme toggle with saved user preference
- SEO metadata and responsive accessible layout
- Sample data throughout for quick editing and customization

## Run locally
```bash
npm install
npm run dev
```

## Build for production
```bash
npm run build
npm run preview
```
